# Решения задач

Здесь находятся решения задач по теме retention.

Каждое решение включает подробные комментарии, чтобы объяснить, как и почему оно работает.

## Пример решения: Retention

1. Сначала мы создаём когорты на основе даты регистрации пользователя.
2. Затем мы считаем активность пользователей по неделям.
3. В конце мы считаем количество вернувшихся пользователей (retained users), группируя по неделям их регистрации и активности.
